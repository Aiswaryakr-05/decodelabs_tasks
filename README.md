# Decodelabs Internship Projects

This repository contains projects completed as part of my Data Analytics Internship at Decodelabs. The projects demonstrate practical skills in data cleaning, exploratory data analysis (EDA), and data visualization using Python.

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- Matplotlib

---

## Project 1: Data Cleaning

### Objective
Prepare a raw e-commerce dataset for analysis by improving data quality and consistency.

### Tasks Performed
- Loaded the dataset using Pandas
- Checked data types and dataset structure
- Identified and handled missing values
- Removed duplicate records
- Corrected data formatting
- Prepared a clean dataset for analysis

### Outcome
A cleaned and structured dataset ready for further analysis.

---

## Project 2: Exploratory Data Analysis (EDA)

### Objective
Analyze the cleaned dataset to discover patterns, trends, distributions, and outliers.

### Analysis Performed
- Examined dataset structure and summary statistics
- Calculated mean, median, count, minimum, and maximum values
- Analyzed product popularity
- Examined order status distribution
- Analyzed referral sources
- Created visualizations using Matplotlib
- Identified potential outliers using box plots
- Summarized key business insights

### Key Findings
- The dataset contains **1,200 records** and **14 columns**.
- No missing values were found after data cleaning.
- **Printer** was the most frequently ordered product.
- **Instagram** was the leading referral source.
- **Cancelled** orders were the most common order status.
- A few high-value outliers were identified in the **TotalPrice** column.
